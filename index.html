About Me

My name is Brad. I was interested in how websites are made, so I started learning web development.

For me, development is purely a hobby. My plan was learn to code & try to replicate various investment tools I'd made for work.

Going from knowing nothing about programming to being able to build things on the web has been a very rewarding process. After squeezing in development time around work and family I've come to the realisation this is what I want to do for a career.
